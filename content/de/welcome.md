Willkommen im digitalen Barrierefreiheitsportal des Großherzogtums Luxemburg. Hier finden Sie eine Reihe von Ressourcen, die Ihnen helfen, Ihre Websites und mobilen Anwendungen für alle zugänglich zu machen, einschließlich
- das [Web Accessibility Assessment Framework (RAWeb 1.1) (EN)](/en/raweb1.1/index.html);
- das [Mobile Application Accessibility Assessment Framework (RAAM 1.1) (EN)](/en/raam1.1/index.html).

Im Rahmen des Barrierefreiheitsprozesses müssen öffentliche Stellen in Luxemburg für jede von ihnen veröffentlichte Website und Anwendung eine Erklärung zur Barrierefreiheit erstellen. [Mit diesem Tool können Sie ganz einfach Ihre Erklärungen zur Barrierefreiheit erstellen](tools/decla.html).

Wenn Sie Fragen haben, können Sie uns gerne [kontaktieren](contact.html).
